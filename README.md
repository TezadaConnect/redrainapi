# Red Rain API

This project is a RESTful API built with Laravel, designed to provide robust authentication and user management features. It leverages Laravel's built-in authentication mechanisms along with Sanctum for API token management.

This Project is intended to be the backend service of Red Rain Game Application.

## Technologies Used

-   Laravel Framework
-   Composer
-   XAMPP / Local Development Server
-   MySQL Database
